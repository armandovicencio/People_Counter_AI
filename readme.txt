# 👥 AI People Counter Pro

Sistema avanzado de conteo de personas y análisis de video con extracción de metadatos.

## 🚀 Características Principales

### 🎯 Detección Avanzada
- **YOLOv8** para detección precisa de personas
- Múltiples tamaños de modelo (n, s, m, l, x)
- ROI (Región de Interés) ajustable
- Configuración de umbral de confianza
- Optimización de frames para procesamiento rápido

### 📊 Análisis Completo
- Conteo de personas por segundo/minuto
- Timeline interactivo con Plotly
- Heatmaps de densidad
- Análisis de tendencias y patrones
- Estadísticas detalladas

### 🔍 Extracción Inteligente
- OCR para fechas y horas visibles
- Detección de coordenadas GPS
- Metadatos técnicos del video
- Información de ubicación con reverse geocoding
- Extracción de texto relevante

### 📁 Exportación Flexible
- CSV con datos de conteo temporal
- JSON con estructura completa
- Múltiples formatos de salida
- Vista previa de datos integrada

## 🛠️ Instalación

### Prerrequisitos
- Python 3.8+
- Tesseract OCR
- FFmpeg

### 1. Clonar e instalar dependencias
```bash
git clone <repository-url>
cd person_counter_app
pip install -r requirements.txt



Estructura del Proyecto

├── app.py
